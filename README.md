# MediaLister

A personal media tracking application for managing movies, TV shows, and games across backlogs and yearly rankings.

## Stack

- **Frontend**: Next.js 14+ (App Router, TypeScript)
- **Styling**: Tailwind CSS + shadcn/ui
- **Database**: Supabase (Postgres + Realtime)
- **ORM**: Prisma (optional)
- **Deployment**: Vercel
- **PWA**: Service Worker + Manifest

## Features

- **Backlogs**: Organize movies, shows, and games by streaming service/platform
- **Rankings**: Yearly ranked lists with ratings and metadata
- **Real-time Sync**: Updates across devices via Supabase Realtime
- **Tags & Filters**: Flexible tagging system for filtering and organizing
- **PWA Support**: Installable app with offline capabilities

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run the development server: `npm run dev`
